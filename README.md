# Computer-Security


## `Prerequisites`
- C++ 20+
- CMake

## `Build and Run`

``` Powershell
cd <project>
```

``` Powershell
cmake -S . -B build -DCMAKE_TOOLCHAIN_FILE=<path to vcpkg>
cmake --build build
````
